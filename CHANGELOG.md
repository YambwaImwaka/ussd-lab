## [0.1.0] - 2025-11-27

### Added

- Core module with USSD app definition and validation
- Simulator module for running USSD sessions
- Testing helpers and assertions for Vitest/Jest
- Comprehensive API documentation
- Getting started guide with examples
- Support for custom screen handlers
- Session state/data management
- Error tracking and reporting
- Session timeout and max screens configuration
- Input validation patterns
- Conditional routing support

### Features

- **Define USSD Flows in Code**: Create screens and routing declaratively
- **Simulate Sessions**: Run and test complete user journeys
- **Validate Apps**: Detect unreachable screens, invalid routes
- **Test Helpers**: Built-in assertions for automated testing
- **Session State**: Store and retrieve data across screens
- **Custom Logic**: Handler functions for validation and routing
- **Error Handling**: Track and report session errors
